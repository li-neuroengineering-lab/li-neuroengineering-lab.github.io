# Li Lab Website Starter

This is a static website starter for GitHub Pages.

## Files

- `index.html` — homepage
- `research.html` — research overview
- `people.html` — current lab members
- `publications.html` — Google Scholar link and selected publication placeholders
- `news.html` — placeholder news page
- `contact.html` — contact and recruiting text
- `assets/css/styles.css` — visual design
- `assets/img/` — images extracted from the supplied PPT materials

## GitHub Pages deployment

1. On GitHub, create a repository named `YOURUSERNAME.github.io`.
2. Upload all files in this folder to that repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and the `/root` folder, then save.
6. Visit `https://YOURUSERNAME.github.io`.

## Important launch checklist

- Replace or confirm all headshots.
- Replace placeholder profile bios.
- Verify office room, phone number, and recruiting language.
- Replace selected publication placeholders with a clean Google Scholar/BibTeX-derived list.
- Add real lab news and project/funding entries.
- Remove any image that should not be publicly posted.
